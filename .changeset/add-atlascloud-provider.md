Add Atlas Cloud as a built-in provider. Atlas Cloud is an OpenAI-compatible gateway (`https://api.atlascloud.ai/v1`) that routes to 300+ models across labs (Claude, GPT, DeepSeek, Qwen, GLM, Llama, and more) behind a single API key. Connect it from the Routing page with an Atlas Cloud API key — model discovery, the `/v1/chat/completions` proxy, and streaming usage reporting all work like the other OpenAI-compatible providers.
